my_tracking module¶

my_tracking.check_elements(message, tgbot)[source]¶

Check HTML changes on tracking page.

Parameters:

  • user – current user.

  • track_name – name of the chosen track.

  • current_track – data of the chosen track, found by track_name.

my_tracking.check_images(message, tgbot)[source]¶

Call function that will check if the image of site have been changed.

Parameters:

  • current_track – current track that user’s chosen.

  • user – id of current user.

  • response – an object that has an information about if image’s changed and how has it been changed.

  • answer_text – text that will be displayed according on if there is a change or no change. Have 2 varieties of displaying.

  • answer_photo – photo that will be shown when there is some changes in tracking site.

my_tracking.check_track(message, tgbot, answer=None)[source]¶

Call function that will show buttons of interaction with chosen tracking.

Parameters:

  • current_track – current track that user’s chosen.

  • user – id of current user.

  • keyboard – inline buttons list.

my_tracking.delete_track(message, tgbot)[source]¶

Delete the chosen track from user’s trackings. Removes from the list, and all the corresponding data - from the folder.

Parameters:

  • user – current user.

  • track_name – name of the chosen track.

  • keyboard – keyboard of inline button.

  • remove_text – the text, that will be written in the message after deleting the tracking.

my_tracking.generate_checking_msg(current_track)[source]¶

Generate message that will display buttons of tracking check (by images or by elements) and delete button. It will also have an info about tracking itself - name, url and time of the last update.

Parameters:

  • converted_time – last update time, converted to format as day.month.year hour:minute:second.

  • keyboard – keyboard of inline buttons of checking and deleting elements + back button.

  • current_track – track that was chosen for interaction.

my_tracking.prestart_show_tracks(bot)[source]¶

Add show_tracks function to bot functionality, which will be first started when the text ‘📂 мои отслеживания’ is written. It also have a logic of interaction between inline keyboard buttons and functions.

my_tracking.show_tracks(message, tgbot)[source]¶

Call function with inline keyboard with all the existent trackings of user.

Parameters:

  • message – message that was received.

  • user – id of current user.

  • keyboard – keyboard with all trackings.

  • num – amount of trackings that user have.
